# ruff

A console e-mail client written in Rust.

## setup
```
# create ~/.ruffrc file
cargo run
```

## .ruffrc example
```
[accounts]
  [accounts.default]
    imap_server = "imap.example.com"
    imap_port = 993
    imap_user = "username"
    imap_pass = "password"
```
